Forms and Input

Input

Bootstrap comes with several simple to use pre designed form controls, text area, checkboxes, radio buttons, and Select lists all have pre set styles in bootstrap. Most will be created by applying the .form-group class to containing elements

Text area uses form-group on the container, and the textarea element with the .form-control class, rows is a variable that you can set.

Forms

The library also contains three form templates, The in-line and verticle are very simple to utilize and can be implemented within a matter of moments. Below is an example of a vertical form, which is the bootstrap default. Forms like these are created by using the "form" element around a div using the "form-group" class, the fields within follow standard form elements.

Alerts

Alerts are very useful for highlighting an area, not necessarily in a pop up format but to bring attention to any single area which needs focus.

alerts are variations on a standard well, with color added to highlight the information within

Success! Indicates a successful or positive action.
Info! Indicates a neutral informative change or action.
Warning! Indicates a warning that might need attention.
Danger! Indicates a dangerous or potentially negative action.
