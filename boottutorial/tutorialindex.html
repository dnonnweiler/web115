Introduction

Information

Bootstrap is a front-end framework that is widely utilized and under continual development, it was developed by Twitter developer Mark Otto and Jacob Thornton. The goal was to develop a universal framework for website development that would contribute to much faster and efficient sites, as well as encourage consistency across platforms. It has continued to be updated to include the latest in CSS and JS capabilities.

Functions

Bootstrap is a one stop shop for many effects that would take a great deal of time if a developer were to write them from scratch. It allows for swift implementation of responsive design by pre coding the standard styles that such an undertaking would generally require. It contains a great deal of customisation libraries, from typography to image shapes and gallery layouts. One of Bootstraps simplest features yet greatest strengths though is the Grid layout, allowing one to swiftly shape a site proportionally to the content while avoiding restrictive positioning and micromanaging. Bootstrap library holds styles for every aspect of front end development, from forms, inputs, navigation, menus, alerts, even aspect ratio and responsive design.
