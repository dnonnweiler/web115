Layout

Tables

Tables in Bootstrap can be assigned styles to make them much easier to lay out and read. Adding the .table class to the table element producces a plain but simple to read table.

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

adding the .table-striped class alternates the row background, for a much easier to read layout.

Wells

The .well class creates a thin rounded border around the element, assigning additional padding and a grey background, this is used to break up information fields on a site so it is easier to read.

Panels

Panels are more advanced than wells and can be used to create tab card style layouts for ease of dissemination between article headings and the information contained within.

the .panel-group when applied to a container, removes the bottom-margin of each panel
There are also multi use panels, using the same variations as the alert panes.

Graph layout

The graph effects or column classes within bootstrap speeds up the arrangement of a developing page by a considerable amount, all columns can add up to any variation of 12, you could have 12 single column div elements or two 6 column. These columns also come in three heights, sm, md, lg, these are set to determine the line height of the content.
To utilize the graph capabilities of Bootstrap, it is a 3 part class, col-"size"-"space" for example

col-sm-2
col-sm-6
col-sm-4
