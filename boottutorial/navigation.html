Navigation

Navbar

The Navbar element within bootstrap is one of its best and easiest to use classes, what used to take extensive javascript and CSS is made much simpler through Bootstrap classes, automatically placing a nav ul as an inline menu. An example of the Navbar funciton is at the top of the page. The class .navbar defines the nav element within the page while .navbar-default selects the behaviour and color, .navbar-inverse uses a black background. applying the .nav and .navbar-nav classes to ul and li elements within a navbar container will behave as an inline box, staying on the same level as well as formatting the apperance of the text.

Dropdown

The navigation bar class allows for easy and effortless implementation of nesting of ul elements to create dropdown menus, this is done by appling the .dropdown class to a li element within the navbar container, then creating a ul using the class .dropdown-menu within the li container.

Pager

Creates a set of arrows to the right and left,, to create the pager object, just place a ul using the class .pager the arrows can be set to the limits of the page wrap by using .previous and .next on the li elements within.
