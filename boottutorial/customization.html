Customization

Glyph Icons

Glyphicons are small sprites that can be used to label and inform users of the purpose of a field or entry. They can be placed outside of a field or, utilizing the span element, be placed within other elements such as to label a search button
The class identifiers for glyph icons are

.glyphicon: which defines the space that a glyph icon will take
.glyphicon-glyphname: which decides what sprite is called. For example .glyphicon-search produces

Badges

Badges are used to place labels or highlight text next to another field using the class .badge you can include any text inside of the element, so it is useful as both a styling implement and an informative element.

Typography

Bootstrap uses a default font size of 14, slightly larger than the standard font in most browsers, this difference extends into "p" and heading elements as well, with the default in bootstrap being slightly larger, with the line height adjusted accordingly, this leads to a very readable website with smooth spaces between paragraphs. There is several custom elements included with bootstrap that will allow for further customisation of the typography of elements.

The "small" element reduces the font size by approximately half and uses a lighter shade to help highlight subtext in a heading.

The mark element can be used to highlight text (default is light yellow)

"abbr" can be used as a simple informative mouseover, especially useful for abbreviating well known names while leaving a title in the element to explain in mouseover. Example WWF

"blockquote" becomes a nesting element in bootstrap, adding margins and special styling to other elements within.
